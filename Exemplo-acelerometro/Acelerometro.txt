# Raspberry pi - Integração com acelerometro e giroscopio

(Referencia: http://blog.bitify.co.uk/2013/11/interfacing-raspberry-pi-and-mpu-6050.html)

## Passos:

### Adicionar linhas abaixo no arquivo sudo vi /etc/modules

i2c-bcm2708
i2c-dev

sudo apt-get install i2c-tools
sudo i2cdetect -y 1

### Conectar o circuito 

### endereço 0x68

sudo i2cget -y 1 0x68 0x75
### retorna 0x68

## Instalar 
sudo apt-get install python-smbus  

Code:
./acelerometro.py
